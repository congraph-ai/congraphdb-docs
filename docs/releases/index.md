## Release Schedule

CongraphDB follows semantic versioning:

- **Major** (0.x.x) — Breaking changes (none yet, still in 0.x)
- **Minor** (x.1.x) — New features, backward compatible
- **Patch** (x.x.2) — Bug fixes, backward compatible
